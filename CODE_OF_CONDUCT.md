# Code of Conduct

## Our Standards

I expect contributors and maintainers to act with respect, professionalism, and restraint. Good community behavior includes:

- Using welcoming, inclusive language
- Focusing feedback on code and ideas, not people
- Being patient with different experience levels
- Respecting privacy and data boundaries

## Unacceptable Behavior

The following are not acceptable in this project:

- Harassment, abuse, or intimidation
- Personal attacks or insulting language
- Discriminatory comments or behavior
- Deliberate disruption of discussion or contributions
- Sharing private information without permission
- **Metric Farming & Contribution Fraud**: Automating pull requests, generating fake or trivial activity, inflating line counts via stacked branches, or gaming open-source programs (such as ELUSOC) to farm points.

## Enforcement

Maintainers may remove, edit, or reject contributions that violate this code of conduct. Repeated or serious violations may result in further restriction from project participation.

## Reporting

If you believe someone has violated this code of conduct, report it to the project maintainers through the repository issue or contact channel used by the project.
